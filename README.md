# openllm-selector

A tool to help researchers pick the right open LLM (Large Language Model) for their study.

## About

Choosing the right open LLM for research can be challenging given the rapidly growing 
landscape of available models. This package provides a curated database of open LLMs 
with key characteristics to help researchers make informed decisions.

## Features (planned)

- Filter and compare open LLMs by characteristics (size, open weights, checkpoints, modality)
- Access foundational papers for each model
- Find recent papers using a specific LLM
- Visualize how models cluster based on their characteristics

## Installation

```bash
pip install openllm-selector
```
